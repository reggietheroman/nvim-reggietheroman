# Neovim Config

Personal Neovim configuration. Single-file setup — all config lives in `init.lua`.

## Key facts

- **Requires Neovim 0.11+** — uses `vim.pack.add` (native package manager). Do not add lazy.nvim, packer, or any external plugin manager.
- `nvim-pack-lock.json` is auto-generated by `vim.pack`. Do not edit manually.

## Plugins

nvim-tree (file explorer), telescope (fuzzy finder), toggleterm (terminal split), nvim-web-devicons, plenary (telescope dependency).

## Conventions

- 2-space indentation, tabs converted to spaces.
- Netrw is disabled — nvim-tree replaces it.
- Comments are encouraged in this config (unlike typical codebases).
- `<leader>e` toggles nvim-tree, `<leader>t` toggles the terminal split.
